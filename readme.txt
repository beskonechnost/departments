Реализовано простое приложение позволяющее просматривать отделы в офисе, добавлять новые, удалять (вместе со всеми сотрудниками) и редактировать их.
В каждом отделе можно просмотреть список сотрудников, удалить сотрудника, добавть новго, отредактировать параметры старого.
Так же можно просмотреть весь список сотрудников всего офиса. Для выборки данных из БД реализованы viwe (данные о департаментах и сотрудниках)
При добавлении и редоктировании данных - валидация происходит на сервере. При добавлении сотрудника валидация частично происходит на клиетне -
в частности такие поля телефон и email. Работа реализована через шаблоны проектирования MVC и Command. Работа с БД реализована через singleton.
Производиться логирование работі с помощю log4j.
